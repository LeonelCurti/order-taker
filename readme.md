# Order-Taker App

> Small business oriented application that allows to manage orders placed by clients. 

### Techs:
- mongodb
- express
- react
- node

### Specs
- Users should be able to:
   * Login, logout and register
   * See price list
   * Make new order
   * See orders made
   * Edit order
- Admin should be able to:
   * Upload main price list
   * Manage clients orders
